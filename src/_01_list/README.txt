List
======

Задание 1
------------
1) При помощи List создайте
перечень товаров из 8-и наименований.
2) Вставьте во вторую позицию перечня
новое наименование.
3) Добавьте к перечню еще одно
наименование товара.


Задание 2
------------
1) При помощи List создайте
перечень товаров из 8-и наименований.
2) Добавьте к перечню еще пару
наименований товаров.
3) Удалите из перечня товар из
6-й позиции перечня.


Задание 3
------------
1) При помощи List создайте
такой перечень имен:
Алина
Виктор
Яна
Дмитрий
Владимир
Борис
Ольга
2) Выведите нумерованый
перечень имен по алфавиту
(А-Я).


Задание 4
-----------
1) При помощи List создайте
перечень товаров из 5-и наименований.
2) При помощи List создайте 2-й
перечень товаров из 7-и наименований.
3) Добавьте 1-й перечень ко 2-му.
4) Выведите нумерованные
начальные перечни и результат их
объединения.


Задание 5
-----------
1) При помощи List создайте перечень
товаров из 5-и наименований.
2) При помощи List создайте 2-й
перечень товаров из 3-х наименований.
3) Добавьте 2-й перечень с 3-й позиции
1-го перечня.
4) Выведите в консоль нумерованные
начальные перечни и результат их
объединения.


Задание 6
------------
1) При помощи List создайте перечень
товаров из 5-и наименований.
2) Проверьте наличие одного из
наименований в перечне.
3) Сообщение о результатах проверки
выведите в консоль.
